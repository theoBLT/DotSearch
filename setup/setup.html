This extension fetches all its time-saving keyboard shortcuts from a centralized AirTable database that you own.

What AirTable API URL can I fetch all your shortcuts from?

To find out,

  • Open Airtable's API page
  • Select the base you've created to host your shortcuts
  • In the API documentation's menu, hit List records
  • Copy the URL shown in the "Example request", but remove everything after the "?"

What AirTable API Key should I use?

To find out, follow this helpful guide. Protip: you can create a separate "API user" and share the base with that user as read-only. That user's API key will only have read access to the base.
