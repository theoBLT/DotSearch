What's the URL to retrieve all records from your airtable base?

To find out,

  • Open Airtable's API page
  • Select the base you've created to host your shortcuts
  • In the API documentation's menu, hit the List recordsitem
  • Copy the URL shown in the "Example request", but remove everything after the question mark (?)

What's your airTable API Key?

To find out, follow this helpful guide. Protip: you can create a separate "API user" and share the base with them as read-only. That user's API key will only have read access to the base.
